How does GhostWatcher run?

Instagram blocks websites from loading outside code (a security rule called Content-Security-Policy). So GhostWatcher can't be a tiny link — it runs as one self-contained piece, directly in your browser. The methods below are built for exactly that. Pick whichever feels easiest. Method 1 is the smoothest.

Three ways to install

All free. All run locally. Choose one.

1

Drag & drop bookmark

Recommended · one click forever

EASIEST

Drag this button up to your bookmarks bar. Don't click it — drag it.

👻 GhostWatcher

↑ drag me to the bookmarks bar ↑

🖱️📷👆
  1. Show your bookmarks bar — press Ctrl+Shift+B (⌘+Shift+B on Mac).
  2. Drag the blue GhostWatcher button onto that bar.
  3. Open instagram.com and log in.
  4. Click your new GhostWatcher bookmark. The overlay opens. ✨

Why drag? It copies the whole bookmark perfectly. Pasting long bookmark text can get silently cut off — the usual reason a bookmarklet "doesn't work."

2

Copy the bookmarklet

If dragging isn't an option (e.g. mobile)

  1. Tap Copy bookmarklet below.
  2. Create a new bookmark (any page), then edit it.
  3. Replace its URL/address with what you copied, and save.
  4. On Instagram, open that bookmark to run GhostWatcher.

⚠️ Some browsers trim very long bookmark URLs when pasted. If it doesn't run, use Method 1 (drag) instead — it never trims.

3

Paste in the console

The classic method · always works

  1. Open instagram.com and log in.
  2. Open DevTools Console — F12 or Ctrl+Shift+J (⌘+Option+J on Mac).
  3. If warned about pasting, type allow pasting and press Enter.
  4. Tap Copy full script, paste into the console, press Enter.

Questions

Is it safe? Do you see my password?▾

GhostWatcher never asks for or sees your password. It runs entirely in your own logged-in browser session and talks only to Instagram. No data is sent anywhere else — there are no servers involved.

Can my account get blocked?▾

Automating actions can violate Instagram's Terms and may trigger temporary action blocks. GhostWatcher paces itself with human-like delays to reduce risk, but you should keep unfollow volumes low and use it at your own risk.

Why isn't there a short one-line command?▾

Instagram's Content-Security-Policy blocks pages from loading external code, so a short link that fetches the script from GitHub gets rejected. The whole tool has to run as one self-contained piece — that's why we use a bookmarklet or a full paste.

Does it work on mobile?▾

It works best on desktop browsers where you have a console and a bookmarks bar. Some mobile browsers support bookmarklets (Method 2), but the experience is smoothest on a computer.
